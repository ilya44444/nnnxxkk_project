# Путь к себе | Individualism Website

## Описание проекта

Это образовательный веб-сайт, созданный для распространения индивидуализма среди молодежи. Сайт предоставляет информативные и познавательные материалы, мотивирующие молодых людей развивать индивидуальность вместо зависимости от моды и социальных сетей.

## Особенности

- 🎨 **Визуально привлекательный дизайн** - современный градиентный дизайн с плавными анимациями
- 📚 **Образовательный контент** - полезная информация об индивидуализме и личностном развитии
- 💡 **Мотивирующие материалы** - вдохновляющие цитаты и практические рекомендации
- 📱 **Адаптивный дизайн** - работает на всех устройствах (десктоп, планшет, мобильный)
- ⚡ **Интерактивные элементы** - плавная навигация, анимации при прокрутке
- 🌟 **Уникальный контент** - принципы индивидуализма, ресурсы для саморазвития

## Разделы сайта

1. **Главная** - героический раздел с призывом к действию
2. **О нас** - что такое индивидуализм и его основные принципы
3. **Философия** - шесть ключевых принципов индивидуализма
4. **Ресурсы** - книги, практики и упражнения для развития
5. **Твой путь** - пошаговый путь к индивидуализму

## Технологии

- HTML5 - семантическая разметка
- CSS3 - современные стили, градиенты, анимации
- JavaScript - интерактивность и динамические эффекты

## Как использовать

1. Откройте файл `index.html` в браузере
2. Навигация работает через меню или плавную прокрутку
3. Изучайте материалы и применяйте принципы в жизни

## Структура проекта

